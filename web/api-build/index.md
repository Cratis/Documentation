# Cratis API reference

The generated .NET API reference for the Cratis client libraries. Pick a library from the
navigation:

- **Chronicle** — the event store clients: the DotNET client, the AspNetCore integration, and the Testing helpers.
- **Arc** — the full-stack CQRS framework and its MongoDB integration.
- **Fundamentals** — the shared building blocks (concepts, types, serialization).

For the TypeScript / JavaScript API (`@cratis/*` packages) and the narrative documentation,
return to the [main documentation site](/).
